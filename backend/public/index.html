API Endpoints

This REST API allows you to manage Todo in your application.

Todo Endpoints

  • GET/api/todos
  • POST/api/todos
  • PUT/api/todos/1
  • DELETE/api/todos/1

How to Use

  • Base URL: http://localhost:4000
  • Content Type: json for POST and PUT requests

HTTP Methods

  • GET - Retrieve data
  • POST - Add new data
  • PUT - Update existing data by ID (change the status of todolist)
  • DELETE - Delete data by ID
